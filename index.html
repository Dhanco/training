Training Plan:

  • Goals
    • Provide a brief overview of Github
    • Give trainee a working knowledge of basic Github design
    • Teach trainee what Github is used for and how to use it efficiently
  • Overview
    • Will first go over what Github is and how it works.
    • Will explain why it is useful and what can be done with it
  • Training Goals and Objectives
    • Github overview
    • What Github is used for
    • Basic useage of Github
    • how to use Github Effectively
  • Assumptions
    • Trainee knows some basic HTML
    • Trainee does not already know to use Github
    • Trainee has working computer software knowledge
  • Training Approach
    • Primarily guide the trainee through hands-on usage of the program
    • Have a set of tasks listed out to guide the trainee in the basic usage of Github
    • Talk them through the steps if necessary and offer guidance
    • Have the traineed create or utilize a sample HTML page to upload as an example of Github usage
  • Training Curricula
    • Basic Description: Describe what Github is and is used for. (5 min)
      • Overview of Github
      • What is it used for
    • Sample Page Creation and Software Usage: Create a sample page and have trainee see software in action. (5-7 min)
      • Go over software
      • Have trainee create sample page or use created one
    • Uploading and Publishing: Have traineed publish their first website with Github (3-5 min)
      • Have trainee upload and merge any files necessary in Github
      • Have trainee view their completed sample webpage
  • Training Schedule: (See Above Times)
  • Training Logistics
    • At least one computer with up to date software that supports Github
    • Training Guide Print Out
    • A work station for the trainee
    • A sample page set up for Github
    • A Github Account
  • Contingencies
    • #1 - Internet is down
    • #2 - Program is not supported
    • #3 - Computer freezes or crashes
    • #4 - Trainee does not know proper code
    • #5 - Github is down or is running into issues uploading
  • Contingency Plan Procedures
    • #1 - Find a new location with stable internet
    • #2 - Download proper applications to view and use Github if possible; Else get another computer
    • #3 - Restart computer and if it still does not work, find a different computer
    • #4 - Teach them very basic HTML or provide a sample page for them to at least show them how Github works if nothing else
    • #5 - Wait until Github stabilizes
